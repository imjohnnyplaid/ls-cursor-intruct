# Language Skull - Cursor Phase Prompts

These prompts are designed to be used one phase at a time with Cursor.

## How to use:
1. Open the phase file you want to build
2. Copy the entire content
3. In Cursor, start a new chat or use Composer
4. Paste the prompt and say "Build this exactly as described"

Follow docs/13_INCREMENTAL_BUILD_PLAN.md strictly.

After each phase, Cursor should report completion with checklist.
